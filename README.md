📚 ParentsSphere: Bridging the Communication Between Parents and Teachers 👨‍👩‍👧‍👦🎓

ParentsSphere is a dual-role educational platform that strengthens parent-teacher collaboration. With separate logins for Parents and Teachers, the app ensures seamless tracking, communication, and student progress monitoring — all in one place.

📌 Project Overview
ParentsSphere is a user-friendly, secure application designed for schools to streamline student-related operations for both parents and teachers. It empowers parents to stay informed and engaged, while enabling teachers to manage daily academic activities efficiently.

🌟 Features & Functionalities
👨‍👩‍👧 Parent Login
📅 View Attendance: Get real-time attendance updates of your child.
📈 View Scores: Access your child’s academic performance.
💸 Fee Payment: Make secure payments for school fees directly through the app.

👩‍🏫 Teacher Login
✅ Take Attendance: Easily record daily student attendance.
🧮 Add/Update Scores: Enter and manage student academic results.
📚 Student History: View comprehensive student performance records.
